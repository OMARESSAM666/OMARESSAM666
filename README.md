## 📌 Overview

This project documents a comprehensive cybersecurity incident response simulation for a segmented enterprise network.  
Six distinct threats were executed against a 100‑employee organisation:

1. **Phishing** – VPN credential theft via social engineering  
2. **Ransomware** – Double extortion with LockBit 3.0  
3. **Remote Access Trojan (RAT)** – Covert C2 channel over HTTPS  
4. **DDoS Attack** – Volumetric attack taking down a public web server  
5. **Insider Threat** – Privileged user exfiltrating 15,000+ records over three weeks  
6. **Man‑in‑the‑Middle** – Public Wi‑Fi credential interception

Each attack includes:
- Technical breakdown & CIA triad mapping
- Full SOC incident report (classification, timeline, containment)
- Root cause analysis and hardening recommendations
- Network topology and segmentation analysis

**All encrypted systems were restored from backup. $0 ransom was paid.**
